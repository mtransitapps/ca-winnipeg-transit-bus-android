Full description:
This app adds Winnipeg Transit buses information to MonTransit.

This app provides the schedule (offline & real-time) as well as news and service alerts from @winnipegtransit and @transitalerts on Twitter.

Winnipeg Transit buses serve Winnipeg in Manitoba, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by Winnipeg Transit.
http://winnipegtransit.com/en/schedules-maps-tools/transittools/open-data/

This application is free and open-source:
https://github.com/mtransitapps/ca-winnipeg-transit-bus-android

This app is not related with Winnipeg Transit.

Permissions:
- Other: required to read real-time bus stop schedules from the Winnipeg Transit Open Data Web Service API and to read news from Twitter
